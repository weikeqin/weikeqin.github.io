(1.3) 表格的使用

使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行：

name | age
---- | ---
LiSi | 12
LiHua |  32
name age
LiSi 12
LiHua 32

为了美观，可以使用空格对齐不同行的单元格，并在左右两侧都使用 | 来标记单元格边界：

|   name  | age |
| ------- | --- |
| LiSi 	  | 12  |
| LiHua   | 32  |
name age
LiSi 12
LiHua 32 
:--- 代表左对齐
:--: 代表居中对齐
---: 代表右对齐
