securecrt配置

发表于 2017-05-23 更新于 2022-05-28 分类于 tools 阅读次数： 本文字数： 1k 阅读时长 ≈ 1 分钟

(1) 修改全局设置

Options –> Global Options –> General –> Default Session –> Edit Default Settings..

(1.1) 设置字符编码为UTF-8

Appearance 找到 Character encoding 选项, 设置为 UTF-8 格式。

(1.2) 设置字体

Apperance –> Fonts ，设置 Normal font 为 Consolas

(1.3) 设置全局缓冲区大小

Terminal –> Emulation –> Scrollback ，Scrollback buffer 默认为500行，可以最大调整到128000行。

(1.4) 设置日志

Terminal --> Log File ，
设置 Log file name 为 D:\SoftWare\SecureCRT_x86_7.1.1.264_en\log\ssh_log\%S_%Y_%M_%D.log

Options 设置 Start log upon connect 和 Append to file

(1.5) 设置每300秒发个回车不掉线

Terminal –> Anti-idle Send String \n every 300 seconds

(2) 设置字符编码为UTF-8

Options –> Session Options –> Appearance 。 找到 Character encoding 选项, 设置为 UTF-8 格式。

(3) 设置背景颜色

Options –> Sessions options –> Terminal –> Emulation， Terminal 下拉列表下选择 Linux，勾选 ANSI Color 复选框、Use color scheme 复选框 。

Options –> Global Options –> Terminal –> Apperance –> Advanced, 选中一个 Color schemes 点击右边的 Edit ，设置 Foreground(前景) 和 Background (背景) 。

(4) 设置会话的缓冲区大小

Options –> Sessions options –> Terminal –> Emulation –> Scrollback ， Scrollback buffer 默认为500行，可以最大调整到128000行。
